# 🔋 Real-Time Battery Monitoring and Protection System for EV

## 📌 Overview
This project is a Battery Management System (BMS) prototype using Arduino UNO that monitors battery parameters like voltage, current, temperature, and humidity in real-time.

## ⚡ Features
- Real-time voltage monitoring
- Current sensing using ACS712
- Temperature monitoring using DHT11
- LCD display output
- Automatic battery protection using relay

## 🛠️ Components Used
- Arduino UNO
- ACS712 Current Sensor
- Voltage Sensor Module
- DHT11 Sensor
- 16x2 LCD (I2C)
- Relay Module
- LM2596 Buck Converter

## ⚙️ Working
- Sensors collect data (Voltage, Current, Temperature)
- Arduino processes data
- Values displayed on LCD
- Relay disconnects battery during unsafe conditions

## 🔮 Future Scope
- IoT-based monitoring
- Mobile app integration
- Machine learning for prediction
